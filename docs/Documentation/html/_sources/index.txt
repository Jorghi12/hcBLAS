HCBLAS Documentation
====================

This is an implementation of `Basic Linear Algebra Subprograms <http://en.wikipedia.org/wiki/Basic_Linear_Algebra_Subprograms>`_, levels 1, 2 and 3  using C++ AMP and optimized for the AMD GPU hardware.
